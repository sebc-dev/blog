## Notes

-   Les versions exactes (patchs spécifiques si non listés) seront celles disponibles et stables au moment de l'initialisation du projet et seront épinglées dans les fichiers de dépendances (`package.json`, `pom.xml` ou `build.gradle`) pour éviter les mises à jour majeures implicites.
-   **Java :** Utilisation de JDK 21 (LTS) pour la production et les builds Spring Boot. JDK 24 (Current) peut être utilisé pour des explorations locales.
-   **Node.js :** Utilisation de Node.js 22 (LTS) pour les environnements de build Astro. Node.js 24 (Current) peut être utilisé pour des explorations locales.
-   Les montées de version futures devront être planifiées et testées rigoureusement, en suivant les recommandations de non-régression et de sécurité.
